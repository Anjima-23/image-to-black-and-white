Processing a large batch of images one by one can be tedious. This Python script automates the transition from full-color images to clean, grayscale (black and white) versions in seconds.
The script utilizes the OpenCV library to handle image data and the OS module to manage files.
features:
 Processes images and saves them to a separate folder (bw_images).
 Handles any number of images in the source folder in one go.
 Uses industry-standard computer vision techniques.
